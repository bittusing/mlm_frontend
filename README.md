# MLM Frontend

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Features

### Public Pages
- Home - Landing page with features
- About - Company information
- Plans - Available investment plans
- Contact - Contact form
- Login - User authentication
- Register - New user registration with referral code

### User Dashboard
- Dashboard - Overview with stats
- Profile - Update personal and bank details
- Buy Plan - Purchase investment plans
- My Plans - View purchased plans
- Income - Income breakdown and transaction history
- Wallet - Balance and withdrawal requests
- My Team - Direct referrals list
- Team Tree - Hierarchical team structure

### Admin Panel
- Dashboard - Admin statistics
- User Management - View, search, block/unblock users
- Plan Management - Create and manage plans
- Withdrawal Management - Approve/reject withdrawals
- Commission Settings - Configure commission percentages
- Reports - Various business reports

## Technology Stack
- React 18
- Redux Toolkit
- React Router v6
- Axios
- React Toastify
- React Icons
- Recharts (for future graphs)
